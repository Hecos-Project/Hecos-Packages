# Themes Pack

Il pacchetto **Themes Pack** estrae le funzionalità estetiche (temi visuali e aspetto dell'interfaccia) di Hecos, originariamente parte del core, rendendole un modulo indipendente gestito tramite HPM.

## Funzionalità
- Cambia l'aspetto globale della WebUI (Cyberpunk, Turquoise, Pirate, Elegant, Rose, ecc.).
- Applica automaticamente il CSS selezionato senza ricaricare la pagina.
- Supporta il tema automatico basato sul sistema operativo.
- Personalizzazione dei singoli widget nella Control Room.
