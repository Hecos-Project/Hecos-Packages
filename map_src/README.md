# GPS Map Plugin

Questo plugin fornisce le funzionalità di geocoding e geolocalizzazione per Hecos. 
Sfrutta le API pubbliche di Nominatim (OpenStreetMap) senza necessità di chiavi API, permettendo all'assistente di convertire nomi di città o indirizzi in coordinate geografiche (lat/lon).

## Funzionalità
- **Geocoding**: Converte testo in coordinate.
- **Home Location**: Recupera la posizione dell'utente dal profilo (città e indirizzo) per fornire risposte localizzate.
- **Supporto LLM**: Strumenti diretti per l'intelligenza artificiale per l'accesso immediato alla cartografia.

## Installazione
Questo modulo è pacchettizzato come HPKG e gestito automaticamente dal Package Manager di Hecos.
