# Hecos Reminder

The **Reminder** module allows Hecos to schedule time-based alerts using natural language and advanced CRON expressions.
It combines a backend AI tool plugin with a beautiful frontend Web UI widget for the Control Room.

## Features
- **Natural Language Parsing:** "Remind me to call John at 15:00"
- **Recurring Reminders:** "Every Monday at 9:00"
- **TTS Alerts:** Hecos speaks out loud when the reminder triggers.
- **WebUI Notifications:** Visual alerts inside the Hecos interface.
- **Control Room Widget:** View, manage, and dismiss upcoming reminders directly from the UI.
