# Webcam Package

This is the autonomous Webcam package for Hecos. It allows the system to capture images from the local webcam or capture the desktop screenshot for vision models.

It operates entirely autonomously and maintains its own configuration file (`webcam_config/webcam.toml`).

## Installation
1. Compile using `Hecos_HPM_Builder`.
2. Install the `.hpkg` file using the Hecos Package Manager.
